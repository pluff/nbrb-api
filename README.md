# Nbrb::Api

The gem is not complete! Only daily exchange rates methods are available

## Installation

Add this line to your application's Gemfile:

    gem 'nbrb-api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nbrb-api
